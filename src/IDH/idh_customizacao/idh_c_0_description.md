# PROTOCOLO CUSTOMIZAÇÃO

Esta feature tem como objetivo trazer a identidade visual dos parceiros para o app IDH.
A customização está separada em 4 grupos:

1. "application" (aplicação)
2. "header" (cabeçalho)
3. "content" (conteúdo)
4. "clear" (limpar)
